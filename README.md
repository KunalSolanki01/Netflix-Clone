# 🎬 Netflix UI/UX Clone

A **Netflix-inspired UI/UX clone** built completely using **HTML and CSS**.  
This project focuses on recreating the look and feel of Netflix’s interface with a clean layout, responsive design, and modern styling — **no JavaScript used**.

---

## 🚀 Features

- 🎥 Netflix-style homepage layout  
- 📱 Responsive design (desktop & mobile-friendly)  
- 🎨 Modern UI with clean typography and spacing  
- 🖼️ Movie banners, cards, and sections  
- 💡 Built using only **HTML5 & CSS3**

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- Flexbox & Grid
- Responsive Design Principles
